Integration

Integrate in any stack

Script tag, React component, or headless async API — pick the integration style that fits your app.

<!-- No bundler needed. Paste before </body> -->
<script
  src="https://app.yourdomain.com/lipsync-sdk.js"
  data-public-id="YOUR_BOT_ID"
  data-position="bottom-right"
  data-theme="#6366f1"
  data-size="md"
  data-start-open="false"
></script>
import { AvatarWidget, useAvatarPlatform } from '@avatar-platform/react';

// Works in Next.js, Vite, Create React App, Remix
export default function Layout({ children }) {
  const { ask } = useAvatarPlatform('YOUR_BOT_ID');

  return (
    <>
      {children}
      <AvatarWidget
        botId="YOUR_BOT_ID"
        position="bottom-right"
        theme="#6366f1"
        startOpen={false}
        compactMobile
      />
    </>
  );
}
// Headless: use your own UI, the agent answers in the background
const result = await AvatarPlatform.ask(
  'YOUR_BOT_ID',
  'What is your refund policy?'
);

console.log(result.answer);
// → "Refunds are available within 7 days of purchase..."

console.log(result.sources);
// → [{ title: "Refund Policy", url: "...", snippet: "..." }]

// Pass sessionId back to keep conversation context
const followUp = await AvatarPlatform.ask(
  'YOUR_BOT_ID',
  'How do I request one?',
  result.sessionId
);

How it works

Three steps to live

Step 01

Upload your knowledge

Upload PDFs, Word docs, images, audio, video, or paste website URLs. Files are chunked with semantic awareness and embedded via Gemini into searchable pgvector storage.

Step 02

Pick a character

Choose from 4 animated Rive avatars — each with full lip-sync, idle animations, and expression states driven by a Rive state machine. BYO .riv file coming soon.

Step 03

Embed anywhere

Copy one script tag. Your agent is live. Visitors chat in 30+ languages with full real-time lip-sync, source citations, and optional lead capture forms.
