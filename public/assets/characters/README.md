# Character Files

Place your 4 Rive character files here:

- `character_1.riv`
- `character_2.riv`
- `character_3.riv`
- `character_4.riv`

All four must use the same artboard ("Character") and state machine ("InLesson")
and expose the same 23 viseme number inputs (100-122) as required by the SDK.

The character names ("Aria", "Kai", "Nova", "Echo") and descriptions are configured
in `backend/routes/projects.js` — adjust `CHARACTERS` there if you want to rename
or add more characters.
