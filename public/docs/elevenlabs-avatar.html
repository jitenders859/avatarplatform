ElevenLabs Avatar

Use ElevenLabs voices instead of the default Gemini TTS engine. ElevenLabs produces more natural, expressive speech — particularly useful for characters with a specific personality or accent.

Overview

By default, AvatarPlatform uses the voice engine built into the Gemini Live API for real-time speech synthesis. When you configure an ElevenLabs voice, the platform routes audio generation through the ElevenLabs TTS API instead.

The lip-sync pipeline is the same regardless of voice engine: audio is analysed in real-time by the hybrid amplitude+phoneme controller, and viseme weights are driven into Rive via the 23-input state machine.

Configuration

Step 1 — Add your ElevenLabs API key

Add the key to your .env file:

ELEVENLABS_API_KEY=your-elevenlabs-api-key

Step 2 — Set the voice ID per project

In the dashboard, open your project → Settings tab → Voice dropdown. ElevenLabs voices appear in the list once your API key is configured. You can also set the voice ID directly via the API:

PATCH /api/projects/:id
Content-Type: application/json
Authorization: Bearer YOUR_JWT

{
  "voice": "el:21m00Tcm4TlvDq8ikWAM"
}

The el: prefix tells the embed SDK to use the ElevenLabs voice engine. The value after the colon is the ElevenLabs voice ID.

How it works

When a visitor speaks to the agent:

  1. The Gemini Live WebSocket processes speech-to-text and generates a text response
  2. The text response is sent to the ElevenLabs TTS API (server-side proxy at /embed/:publicId/tts)
  3. The audio is streamed back to the browser in chunks
  4. The hybrid lip-sync controller analyses each audio chunk in real-time and drives the Rive state machine

Latency tradeoffs

Gemini Live (default)ElevenLabs
Latency ~300ms end-to-end (fully streaming) ~600–900ms (TTS generation adds a round trip)
Voice quality Good — natural but sometimes robotic on long outputs Excellent — highly expressive, consistent
Language support 30+ languages auto-detected 29 languages, specified per request
Cost Included in Gemini Live pricing ElevenLabs character credits billed separately
💡

Recommendation: Use Gemini Live voices for the lowest latency. Switch to ElevenLabs when voice personality matters more than response speed — for example, a brand mascot or a product demo agent.

ElevenLabs streaming

AvatarPlatform uses ElevenLabs' streaming TTS endpoint (/v1/text-to-speech/:voiceId/stream) with optimize_streaming_latency=3 to minimise the first-byte delay. Audio is passed to the Web Audio API and analysed with a real-time FFT analyser — the same pipeline used for Gemini Live audio.

⚠️

ElevenLabs requires a server-side proxy because the API key must never be exposed to the browser. The proxy endpoint is already built into the embed routes — no extra configuration is needed beyond setting ELEVENLABS_API_KEY.
